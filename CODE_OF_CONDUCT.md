# Code of Conduct

freepalette follows a concise Contributor Covenant-style code of conduct.

## Our Pledge

We pledge to make participation in this project a respectful and harassment-free
experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good faith, while still being clear about technical concerns.
- Accept feedback gracefully.
- Focus on what is best for the project and its users.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private information without permission.
- Disruptive conduct that prevents useful collaboration.
- Any behavior that maintainers reasonably consider inappropriate in a
  professional open-source setting.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict
participation when behavior violates this code of conduct.
